# SCT_WD_3 - Quiz Game Application

## 📌 Overview

This project was developed as part of the SkillsCraft Technology Web Development Internship.

The goal of this task was to build an interactive quiz application that presents multiple-choice questions, evaluates answers, and displays a final score along with performance analytics.

## 🚀 Features

- Interactive Quiz Interface
- Multiple Choice Questions
- Score Tracking
- Performance Analysis
- Correct & Incorrect Answer Count
- Quiz Completion Screen
- Local Leaderboard
- Dark / Light Theme Toggle
- Sound Controls
- Responsive Design
- Smooth User Experience

## 🏆 Result Analytics

The application displays:

- Final Score Percentage
- Correct Answers
- Incorrect Answers
- Performance Message
- Leaderboard History

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
